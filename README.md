# 🐍 農莊貪食蛇大作戰 (Ultimate Snake Game)

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)
![Vanilla JS](https://img.shields.io/badge/JavaScript-F7DF1E?style=for-the-badge&logo=javascript&logoColor=black)


這是一款用純 HTML5, CSS3 跟 Canvas API 硬刻出來的貪食蛇。不只保留了以前按鍵盤玩貪食蛇的經典回憶，還塞了一堆瞎趴的聲光特效、螢幕震動、雙人對戰，甚至還有可以買造型的商店系統。

 ✨ 遊戲亮點
🎮 兩種模式任選
單人 vs 電腦 (1P vs AI)：挑戰會不斷加速的「智慧金色電腦蛇」，看你能活多久。
雙人對戰 (1P vs 2P)：標準的友情破壞機模式！沒有電腦干擾，誰的頭先撞到牆或撞到對方的身體，就直接出局。

🛒 媲美課金手遊的造型系統（但不用錢）
遊戲內努力吃金幣 (💰)，可以解鎖 8 款史詩級造型。
從基礎的 🌈 彩虹漸層，到中二度爆表的 🦖 飛天翼龍、🦄 爆裂天馬、🌌 星空宇宙都有。
 重點是：頂級造型自帶動態粒子特效跟專屬頭飾。
 
 莫名其妙的極致打擊感 (Juice & Polish)
   螢幕震動 & 打擊停頓 (Hitstop)：吃到強力道具或撞牆的瞬間，保證視覺回饋直接拉滿。
   復古合成音效：沒有外掛任何音檔，全靠 Web Audio API 即時合成 8-bit 音效（吃食物、死亡、花錢的音效全都有）。
   動態呼吸背景：農莊草地背景會隨時間微微脈動，告別死氣沉沉的靜態畫面。

⚡ 搞事專用的道具系統**
   場上隨機生成：冰凍、加速、無敵、減速、還有大金幣，讓戰局變數更多。

🕹️ 支援多種輸入設備**
   鍵盤 (WASD / 方向鍵)、滑鼠點擊，甚至接上 **遊戲手把 (Gamepad)** 也能無縫遊玩。

 🚀 怎麼跑起來？

這是一個超乾淨的純前端專案，**不用裝 Node.js，不用 npm install，也不用起 Server**。
